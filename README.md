# WordRecap / WordBoost

A beautiful AI-powered vocabulary builder with spaced repetition, streaks, badges, and Word of the Day.

## Features
- Add words with AI-generated definitions & examples
- Smart practice sessions (MCQ, Fill-in, Flashcards)
- Spaced repetition system
- Daily streak tracking
- Leveling & XP system
- Badges & achievements
- Word of the Day

## How to Run

1. Clone the repo
2. `npm install`
3. `npm run dev`

**Note**: This is a frontend-only React app that uses `window.storage` (you may need to polyfill or adapt for normal localStorage). It also requires an Anthropic API key for full AI features.
